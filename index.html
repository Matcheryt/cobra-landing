Cobra Bot

Cobra Bot is an open source bot with several features
It features music, steam, weather, COVID-19, Oxford dictionary, and more!

Add to server Commands





Commands

General
-help - Shows a link to Cobra's website
-random (minimum number, maximum number) - Gets a random number between minNumber and maxNumber
-usinfo (@User) - Shows info about the mentioned user
-clean (number of messages) - Cleans messages from chat with specified number
-poll (question) - Asks a question on the chat
-lmgtfy (text) - Creates a lmgtfy link with text specified
-dict (word) - Returns the definition of the specified word
-steam (id) - Shows Steam profile Info for a specific SteamID
-covid (country) - Shows COVID19 data for specified country
-weather (city) - Shows current weather for specific city
Music
-play (song name) - Plays song specified
-pause - Pauses music playback
-resume - Resumes music playback
-stop - Stops audio stream and makes bot leave channel
-queue - Lists queued songs
-skip - Skips current song
-remove (queue index) OR (queue start index, queue end index) - Removes songs from queue at index, or
removes songs from start index to end index
-shuffle - Shuffles queue
-lyrics - Shows lyrics for current song

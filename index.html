Cobra Bot

Cobra Bot is an open source bot with several features
It features music, steam, weather, COVID-19, Oxford dictionary, and more!

Add to server Commands





Commands

    General
    -help

    Shows a link to Cobra's website

    -random [minimum Number, maximum Number]

    Gets a random number between minimum Number and maximum Number

    -usinfo [@User]

    Shows info about the mentioned user

    -poll [question]

    Asks a question on the chat

    -lmgtfy [query]

    Creates a lmgtfy link with the specified query

    -dict [word]

    Returns the definition of the specified word

    -steam [account]

    Shows Steam profile info for a specified account

    -covid [country]

    Shows COVID19 data for specified country

    -weather [city]

    Shows current weather for specified city

    Music
    -play [song]

    Plays song specified

    -pause

    Pauses music playback

    -resume

    Resumes music playback

    -stop

    Stops audio stream and makes bot leave channel

    -queue

    Lists queued songs

    -skip

    Skips current song

    -remove [index] OR [start index, end index]

    Removes songs from queue at index, or removes songs from start index to end index

    -shuffle

    Shuffles queue

    -lyrics

    Shows lyrics for current song

    Moderation
    -ban [user] [(optional) prune days] [(optional) reason]

    Bans specified user from the server with optional reason and prune days

    -kick [user] [(optional) reason]

    Kick specified user with optional reason

    -clean [number of messages]

    Cleans messages from chat with specified number (Requires ManageMessages permission)

    -role [user] [+ or - (+ adds role, - removes role)] [role name]

    Adds or removes specified role from user

    -prefix [new prefix]

    Changes bot's prefix. To reset the prefix, use -prefix default

    -setwelcome [#textChannel]

    Sets the channel in which user join/leave messages will appear
